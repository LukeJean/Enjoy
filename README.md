# 享授
## 项目简介
某高校大创项目
## 基本使用方法
1. 数据库用户名密码设置在`config.php`中
2. 创建一个新的数据库，依次执行以下代码
```
use databasename;
exit
source .sql文件夹的路径
```
3. 相关图片放在img文件夹下，相关视频放在video文件夹下，相关课件放在PPT文件夹下
## 文件结构
css    ->存放文件css文件
       ->其中的font文件夹存放字体文件，tubiao存放图标文件
img    ->存放视频封面图片
js     ->存放js代码
PPT    ->存放课件
video  ->存放视频
